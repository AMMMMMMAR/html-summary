What is Metadata?

In HTML, the meta tag is used to provide metadata about the HTML document. Metadata is information about the document itself, not the content it displays.

name="description" indicates that the metadata being provided is a description of the webpage.

content="This is a brief description of the webpage content." is the actual content of the description.

List of Metadata:

  1. Character Encoding: Specify the character encoding of the document.
  2. Viewport Settings: Control the width and initial scale of the viewport for responsive design.
  3. Description: Provide a brief description of the content for search engines.
  4. Keywords: Optionally provide a list of keywords related to the content.
  5. Author: Specify the author of the document.
  6. Viewport Configuration: Control aspects of the viewport like width, initial scale, and whether user scaling is allowed.
  7. Robots: Direct search engine crawlers on how to index or not index the page.
  8. Refresh: Automatically refresh or redirect the page after a specified time.

Open Graph meta tags are used to control how URLs are displayed when shared on social media platforms.
They provide information such as the title, description, image, and type of content.
Here are some common Open Graph meta tags you can use:

  • <meta property="og:title" content="Page Title" />
  • <meta property="og:description" content="Page description." />
  • <meta property="og:image" content="Image URL" />
  • <meta property="og:url" content="Page URL" />
  • <meta property="og:type" content="website" />
